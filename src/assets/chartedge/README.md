# ChartEdge brand assets

- **Runtime (recommended):** `ChartEdgeMark` and `ChartEdgeAttribution` in `src/branding/` — theme-aware and bundled with the library.
- **Static SVG (docs / sites):**
  - `chartedge-mark-light.svg` — for light backgrounds
  - `chartedge-mark-dark.svg` — for dark backgrounds

These are **not** copied into `dist/` by default (`package.json` only ships `dist/`). Copy from this folder into your app or CDN if you need raw files.
